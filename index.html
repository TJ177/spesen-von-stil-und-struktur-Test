App installieren / Hilfe

Android:
1. Link aus der Nachricht kopieren
2. Chrome öffnen

iPhone:
1. Link in Safari öffnen
2. Unten auf Teilen tippen
3. „Zum Home-Bildschirm“ auswählen
4. Hinzufügen drücken

PC:
Link einfach im Browser öffnen und mit demselben Konto anmelden.
